# Team Phoenix Bot 🤖🏆

An all-in-one robot built during the 7-day Robotics Workshop at Robotics Club, Pashchimanchal Campus (WRC). Team Phoenix went unbeaten in the 1v1 speed competition among 10 teams.

## Modes
- Line Following — follows a path using IR sensors
- Wall Following — maintains distance from wall using ultrasonic sensor
- Manual Control — controlled manually

## Components
- ESP32
- IR Sensors
- HC-SR04 Ultrasonic Sensor
- L298N Motor Driver
- DC Motors
- Chassis
- Battery
